Ajax cache clearing(IE)

In some case, the IE will cache the data from last get request, so that the browser cannot fetch the latest informaton from service end.
It works fine in Chrome so no need for Chrome.

To solve this issue we have two ways:

1, Disable the cache in ajax request Header

2, Disbale the cahce in ajax get request with the updated timestamp

VMware install CentOs with Nginx

1, Install gcc.
yum install -y gcc

2, Install perl library, the http module of nginx use pcre to parse regular expression.
yum install -y pcre pcre-devel

3, Install zlib, zlib provides various zip and upzip ways, nginx use zlib to gzip the http packages.
yum install -y zlib zlib-devel

4, Install openssl
yum install -y openssl openssl-devel

5, Instal wget
yum install -y wget

6, Download nginx
wget http://nginx.org/download/nginx-1.9.9.tar.gz

7, Tar the package
tar -zxvf nginx-1.9.9.tar.gz

8, Install

9, Startup the service
/nginx-1.9.9/sbin/nginx

10, If you encounter the issue with the directory of error log, create one folder with a dummy error.log

11, Check the status
ps -ef | grep nginx

12, ifconfig to check the ip address

13, Use the host window system to access the address with the ip address in browser.
