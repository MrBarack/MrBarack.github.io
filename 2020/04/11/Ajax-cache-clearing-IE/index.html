Ajax cache clearing(IE)

In some case, the IE will cache the data from last get request, so that the browser cannot fetch the latest informaton from service end.
It works fine in Chrome so no need for Chrome.

To solve this issue we have two ways:

1, Disable the cache in ajax request Header

2, Disbale the cahce in ajax get request with the updated timestamp
